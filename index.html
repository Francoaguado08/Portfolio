Proyectos

• Sistema de Gestión Bancaria

El sistema desarrollado es una aplicación web de gestión bancaria destinada a un único banco, que contempla dos tipos de usuarios: administrador del banco y clientes. Está diseñado bajo una arquitectura en capas utilizando Java (Servlets, JSP y JDBC) y una base de datos MySQL, promoviendo una clara separación de responsabilidades y escalabilidad. El sistema permite al administrador del banco gestionar clientes y cuentas, aprobar o rechazar préstamos solicitados por los clientes y acceder a reportes estadísticos. El cliente, por su parte, puede consultar sus datos personales, visualizar el historial de movimientos de sus cuentas, realizar transferencias, solicitar préstamos y pagar las cuotas desde sus cuentas.
